# T06: Accés remot. Escriptori remot (RDP) (tasca individual)

## Guies de Suport Gràfic (PoC)

### Context
En la tasca anterior vam treballar amb administració remota de servidors mitjançant **SSH**. Però, què passa quan:
- El servidor és **Windows Server amb interfície gràfica**?
- Un client ens diu: *"No em funciona el programa X"* o *"M'ha sortit un error a la pantalla"*?

Com a consultors d'**EverPia**, no només gestionem backend: també donem **suport directe a l'usuari final**. Això implica:
- Veure el que veu l'usuari.
- Controlar el seu ratolí i teclat.

### Objectiu
Crear **documentació oficial** per a futurs becaris sobre **suport remot gràfic**. Farem una **Prova de Concepte (PoC)** interna amb màquines virtuals per establir les guies d'ús.

### Tecnologia Clau
- **RDP (Remote Desktop Protocol)**:
  - Estàndard de Microsoft per administrar equips Windows.
  - També disponible en entorns GNU/Linux (p. ex. Gnome), permetent connexió a equips com **Zorin** igual que a **Windows 11**.

### Requisits de la Documentació
- Ha de ser **extremadament clara**.
- Pensada per situacions crítiques: un client nerviós espera una solució ràpida.
- La rapidesa i eficàcia en la connexió defineixen la **qualitat del servei**.
